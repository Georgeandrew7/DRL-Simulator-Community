Welcome to DRL Simulator Community Setup!

This installer will set up everything you need to play DRL Simulator 
offline after the official servers were shut down.

WHAT THIS INSTALLS:
━━━━━━━━━━━━━━━━━━━
• Mock Backend Server - Replaces the defunct api.drlgame.com
• BepInEx Mod Framework - Enables plugin loading
• SSL Bypass Plugin - Allows self-signed certificates
• License Bypass Plugin - Enables offline play
• Utility Tools - Binary patcher, extractors, diagnostics

REQUIREMENTS:
━━━━━━━━━━━━━
• Windows 10 or later (64-bit)
• Python 3.8 or later (for the mock server)
• DRL Simulator installed via Steam
• Administrator privileges (for hosts file modification)

WHAT HAPPENS DURING INSTALL:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Files are copied to your chosen installation directory
2. Your hosts file is modified to redirect api.drlgame.com to localhost
3. BepInEx is installed into your game directory (optional)
4. Windows Firewall rules are added (optional)
5. Start Menu and Desktop shortcuts are created
