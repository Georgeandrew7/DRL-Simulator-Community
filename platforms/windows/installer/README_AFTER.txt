GETTING STARTED:

1. Use "Start Offline Mode" from the Start Menu or Desktop
2. Wait for the mock server to start (a terminal window will open)
3. Launch DRL Simulator from Steam as normal
4. Enjoy offline play!

FIRST TIME SETUP:
━━━━━━━━━━━━━━━━━
If this is your first installation, you may need to:

• Install Python packages (if not done automatically):
  Open Command Prompt and run:
  pip install aiohttp requests cryptography

• Install BepInEx manually (if auto-install failed):
  1. Download BepInEx from https://github.com/BepInEx/BepInEx/releases
  2. Extract to your DRL Simulator folder
  3. Run the game once to generate config files
